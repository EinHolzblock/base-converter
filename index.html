Base Legend

Base 2 (Binary): The language of computers (on/off).
Base 6 (Senary): Efficient for finger counting (6 per hand).
Base 12 (Duodecimal): Highly divisible; used in clocks and dozens.
Base 20 (Vigesimal): Ancient Maya system; based on fingers and toes.
Base 36: Max standard JS base; uses digits 0-9 and A-Z[cite: 1, 2].
