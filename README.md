# Base Converter 🔢

A simple, browser-based utility for real-time number system conversion.

## 🚀 Features
*   **Live Conversion:** Updates all fields instantly as you type.
*   **Bi-directional:** Input values in Decimal, Hex, or Binary—the rest follow.
*   **Minimalist Design:** Clean UI with a focus on usability and speed.
*   **GitHub Pages Ready:** Zero-config deployment.

## 🛠️ Tech Stack
*   **HTML5**
*   **CSS3** (Responsive Layout)
*   **JavaScript** (ES6+ Logic)

## 📖 How to Use
1.  Open the `index.html` file in any modern browser.
2.  Enter a value into any of the input fields.
3.  The tool automatically calculates the equivalent values in other bases using:
    *   `parseInt(value, base)` for parsing.
    *   `number.toString(base)` for output.

## 🌐 Deployment
To host this yourself:
1.  Push `index.html` to a GitHub repository.
2.  Go to **Settings > Pages**.
3.  Select the **main branch** as the source and click **Save**.
